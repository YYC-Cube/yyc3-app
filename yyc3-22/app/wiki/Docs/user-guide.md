# 用户指南

## 概述

本文档提供了邮件系统的完整用户指南，帮助用户快速上手和充分利用系统的各项功能。无论您是新用户还是有经验的用户，本文档都将为您提供有价值的参考。

## 系统登录

### 访问登录页面

1. 打开浏览器，输入系统URL：`https://mail.example.com`
2. 系统将自动重定向到登录页面

### 登录步骤

1. 在登录页面输入您的邮箱地址和密码
2. 如果启用了多因素认证（MFA），请按照提示输入验证码
3. 点击「登录」按钮

### 忘记密码

1. 在登录页面点击「忘记密码？」链接
2. 输入您注册时使用的邮箱地址
3. 点击「发送重置链接」按钮
4. 检查您的邮箱，按照邮件中的说明重置密码

## 界面介绍

### 主界面布局

系统主界面由以下几个主要部分组成：

1. **顶部导航栏**：包含用户头像、设置、通知、搜索框等
2. **左侧菜单栏**：包含邮件文件夹、联系人、日历等功能入口
3. **中间内容区**：显示邮件列表或当前选中的邮件内容
4. **右侧边栏**：显示联系人信息、附件预览等（可折叠）

### 左侧菜单栏说明

- **收件箱**：显示所有收到的邮件
- **已发送**：显示您发送的所有邮件
- **草稿箱**：存储未发送的草稿邮件
- **垃圾邮件**：显示系统识别的垃圾邮件
- **已删除**：显示已删除的邮件
- **标签文件夹**：自定义标签的邮件分类
- **联系人**：访问您的联系人列表
- **日历**：查看和管理日程安排
- **任务**：创建和跟踪任务

## 邮件管理

### 撰写新邮件

1. 点击界面左上角的「撰写」按钮
2. 在弹出的撰写窗口中：
   - 在「收件人」字段输入收件人邮箱（支持从联系人中选择）
   - 可选择性地添加「抄送」和「密送」收件人
   - 输入邮件主题
   - 在正文区域撰写邮件内容
   - 使用工具栏格式化文本、添加链接、图片等
3. 点击「发送」按钮发送邮件，或点击「保存草稿」保存为草稿

### 添加附件

1. 在撰写邮件时，点击工具栏中的「附件」图标
2. 选择要上传的文件，或拖放文件到撰写区域
3. 上传完成后，附件将显示在邮件下方
4. 可点击附件旁的「x」按钮移除不需要的附件

### 设置邮件优先级

1. 在撰写邮件时，点击右上角的「更多选项」（三个点）
2. 选择「优先级」，然后选择「高」、「普通」或「低」
3. 优先级将在收件人收到的邮件中标明

### 邮件签名设置

1. 点击顶部导航栏的用户头像，选择「设置」
2. 在左侧菜单选择「邮件设置」→「签名」
3. 输入您的签名内容，可使用富文本格式
4. 选择是否在所有新邮件和回复中自动添加签名
5. 点击「保存」按钮

## 阅读邮件

### 基本阅读操作

1. 在左侧菜单选择邮件文件夹（如收件箱）
2. 在邮件列表中点击要阅读的邮件
3. 邮件内容将显示在右侧区域
4. 如需全屏阅读，点击邮件右上角的「最大化」按钮

### 标记已读/未读

1. 在邮件列表中，点击邮件左侧的圆形图标可切换已读/未读状态
2. 或在选中邮件后，点击顶部工具栏中的「标记为已读」/「标记为未读」按钮
3. 也可右键点击邮件，选择相应选项

### 标记重要/星标

1. 在邮件列表中，点击邮件左侧的星形图标可添加/移除星标
2. 或在选中邮件后，点击顶部工具栏中的「添加星标」/「移除星标」按钮
3. 星标邮件可在左侧菜单的「星标」文件夹中查看

### 邮件搜索

1. 在顶部导航栏的搜索框中输入搜索关键词
2. 按下回车键或点击搜索图标
3. 搜索结果将显示在邮件列表区域
4. 使用高级搜索选项可按发件人、主题、日期等条件筛选

## 邮件组织

### 创建文件夹/标签

1. 在左侧菜单的「文件夹」区域右键点击
2. 选择「新建文件夹」
3. 输入文件夹名称，选择父文件夹（可选）
4. 点击「确定」按钮

### 移动邮件到文件夹

1. 在邮件列表中选中要移动的邮件
2. 点击顶部工具栏中的「移动到」按钮
3. 从下拉菜单中选择目标文件夹，或选择「新建文件夹」
4. 也可拖拽邮件到左侧菜单的目标文件夹

### 使用过滤器自动分类

1. 点击顶部导航栏的用户头像，选择「设置」
2. 在左侧菜单选择「邮件设置」→「过滤器」
3. 点击「新建过滤器」按钮
4. 设置过滤条件（如发件人、主题关键词等）
5. 设置操作（如移动到特定文件夹、标记星标等）
6. 点击「保存」按钮

## 联系人管理

### 查看联系人列表

1. 在左侧菜单点击「联系人」
2. 联系人列表将显示在主界面
3. 使用搜索框可快速查找联系人
4. 使用排序选项可按姓名、公司等排序

### 添加新联系人

1. 在联系人界面点击「新建联系人」按钮
2. 填写联系人信息（姓名、邮箱、电话、公司等）
3. 点击「保存」按钮

### 从邮件添加联系人

1. 打开邮件，将鼠标悬停在发件人/收件人名称上
2. 点击弹出窗口中的「添加到联系人」按钮
3. 完善联系人信息（如果需要）
4. 点击「保存」按钮

### 创建联系人组

1. 在联系人界面点击「新建组」按钮
2. 输入组名称
3. 从联系人列表中选择要添加到组的联系人
4. 点击「保存」按钮
5. 发送邮件时可直接选择该组作为收件人

## 日历功能

### 查看日历

1. 在左侧菜单点击「日历」
2. 默认显示月视图，可切换为周视图或日视图
3. 系统将显示所有已添加的日程

### 创建新事件

1. 在日历界面点击「新建事件」按钮，或直接点击日历中的特定日期和时间
2. 填写事件信息（标题、时间、地点、描述等）
3. 添加参与者（可选）
4. 设置提醒（可选）
5. 点击「保存」按钮

### 发送会议邀请

1. 创建新事件时，在「参与者」字段添加与会者邮箱
2. 添加会议地点和描述
3. 点击「发送邀请」按钮
4. 系统将向所有参与者发送邮件邀请

### 导入/导出日历

1. 点击日历界面右上角的「设置」图标
2. 选择「导入/导出」
3. 按提示上传iCal文件导入日历，或导出当前日历为iCal文件

## 任务管理

### 查看任务列表

1. 在左侧菜单点击「任务」
2. 任务列表将显示在主界面，按截止日期排序

### 创建新任务

1. 在任务界面点击「新建任务」按钮
2. 输入任务标题、描述、截止日期、优先级等信息
3. 选择任务分类（可选）
4. 添加子任务（可选）
5. 点击「保存」按钮

### 更新任务状态

1. 在任务列表中，点击任务左侧的复选框可标记任务为完成/未完成
2. 或打开任务详情，更新状态为「进行中」、「已完成」等
3. 拖拽任务可调整优先级

## 系统设置

### 个人资料设置

1. 点击顶部导航栏的用户头像，选择「设置」
2. 在左侧菜单选择「个人资料」
3. 更新您的姓名、头像、邮箱等信息
4. 点击「保存」按钮

### 密码修改

1. 在设置界面左侧菜单选择「安全」→「密码」
2. 输入当前密码和新密码
3. 确认新密码
4. 点击「保存」按钮

### 通知设置

1. 在设置界面左侧菜单选择「通知」
2. 调整系统通知、邮件通知、桌面通知等设置
3. 为不同类型的事件配置通知方式
4. 点击「保存」按钮

### 快捷键设置

1. 在设置界面左侧菜单选择「快捷键」
2. 查看默认快捷键说明
3. 根据需要自定义快捷键
4. 点击「保存」按钮

## 高级功能

### 邮件模板

1. 在撰写邮件界面，点击工具栏中的「模板」图标
2. 选择已保存的模板或创建新模板
3. 编辑模板内容（如果需要）
4. 使用模板发送邮件

### 自动回复

1. 在设置界面左侧菜单选择「邮件设置」→「自动回复」
2. 启用自动回复功能
3. 设置自动回复的时间段
4. 编写自动回复内容
5. 选择是否只对联系人发送自动回复
6. 点击「保存」按钮

### 邮件召回

1. 在「已发送」文件夹中找到要召回的邮件
2. 点击邮件右上角的「更多选项」（三个点）
3. 选择「召回邮件」
4. 系统将尝试召回邮件（仅当收件人尚未阅读时有效）

### 邮件归档

1. 选中要归档的邮件
2. 点击顶部工具栏中的「归档」按钮
3. 邮件将从收件箱中移除，但可在「归档」文件夹中查看

## 故障排除

### 常见问题

1. **无法登录系统**
   - 检查您的邮箱和密码是否正确
   - 确认您的网络连接正常
   - 尝试清除浏览器缓存和Cookie
   - 如果启用了MFA，确认验证码输入正确

2. **无法发送邮件**
   - 检查收件人邮箱地址格式
   - 确认附件大小未超过限制
   - 检查网络连接
   - 查看系统通知是否有错误提示

3. **未收到预期邮件**
   - 检查垃圾邮件文件夹
   - 确认发件人邮箱地址正确
   - 检查邮件过滤器设置
   - 联系管理员确认系统是否正常运行

## 提示与技巧

1. **使用键盘快捷键**：熟悉并使用快捷键可以显著提高工作效率
2. **创建过滤器**：设置智能过滤器自动管理邮件，减少手动操作
3. **使用标签**：为邮件添加自定义标签，便于分类和查找
4. **定期归档**：将不再需要但仍需保留的邮件归档，保持收件箱整洁
5. **设置合理的通知**：调整通知设置，避免不必要的干扰
6. **利用搜索功能**：使用高级搜索选项快速定位所需邮件
7. **备份重要邮件**：定期导出重要邮件，以防数据丢失

## 术语表

- **MFA**：多因素认证，提供额外的安全验证层
- **IMAP**：Internet消息访问协议，允许从多个设备访问邮件
- **SMTP**：简单邮件传输协议，用于发送邮件
- **草稿**：未发送的邮件，可稍后继续编辑
- **归档**：将邮件移至长期存储区域，但不删除
- **星标**：标记重要邮件的功能
- **过滤器**：根据预设规则自动处理邮件的功能
- **签名**：自动添加到邮件末尾的个性化文本
- **自动回复**：当用户不在时自动发送的回复消息
