# 常见问题解答 (FAQ)

## 问题分类

### 部署相关问题

**Q: 部署过程中遇到端口占用错误怎么办？**
A: 检查系统中是否已有其他服务占用了相同的端口，可通过以下命令查看：

```bash
lsof -i :端口号
```

然后选择停止占用端口的服务，或修改配置使用不同的端口。

**Q: Docker 容器启动失败的常见原因？**
A: 常见原因包括：

* 环境变量配置错误
* 端口映射冲突
* 依赖服务未正常启动
* 数据卷挂载权限问题
请查看容器日志获取详细错误信息。

### 功能使用问题

**Q: 如何创建自定义邮件模板？**
A: 在管理面板中，导航至"模板管理" -> "创建模板"，填写模板名称、主题和内容，支持使用变量占位符。

**Q: 如何配置邮件发送的SMTP服务器？**
A: 在系统配置中，找到"邮件设置"，填写SMTP服务器地址、端口、用户名和密码，保存后系统将使用指定的SMTP服务器发送邮件。

### 技术问题

**Q: 如何排查邮件发送失败的问题？**
A: 首先检查系统日志中的错误信息，然后验证：

1. SMTP配置是否正确
2. 网络连接是否正常
3. 接收方邮箱是否有效
4. 邮箱是否被标记为垃圾邮件

**Q: 系统性能优化建议？**
A:

* 合理配置Redis缓存，减轻数据库压力
* 优化数据库索引
* 考虑使用连接池管理数据库连接
* 对大型操作实现异步处理
