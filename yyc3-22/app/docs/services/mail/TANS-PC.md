# YYC3 NAS 介绍

## YYC3 NAS=铁威马F4-423 NAS

YYC3 NAS（铁威马F4-423）是一款功能强大的 NAS 设备，提供稳定的存储和文件共享功能。它采用 Intel Atom 处理器，支持 Docker 容器化部署，非常适合运行需要数据库支持的应用。TANS PC：TeraMaster 公司精心打造的 TNASPC，是一款专为简化,TNAS 使用体验而设计的多功能集成化电脑桌面助手。它拥有快速定位与连接 TNAS 设备的能力，同时能够辅助用户轻松登录并进行基础配置。此外，TNASPC还提供了远程访问、WOL网络唤醒以及电脑数据备份与同步等实用功能。

## YYC3 NAS（F4-423）已安装启用应用、工具

- TANS PC：TeraMaster 公司精心打造的 TNASPC，是一款专为简化,TNAS 使用体验而设计的多功能集成化电脑桌面助手。它拥有快速定位与连接 TNAS 设备的能力，同时能够辅助用户轻松登录并进行基础配置。此外，TNASPC还提供了远程访问、WOL网络唤醒以及电脑数据备份与同步等实用功能

- Docker Manager :DockerManager是一个具有图形界面，能提供良好用户体验的 DocKer 管理工具。DockerManager 中集合了镜像注册表，并以图形界面的方式实现镜像查询，下载、与container 的安装、配置等丰富的功能;Docker Hub 中上百万的镜像为您的TNAS 提供了多种可能以满足您的业务需要。

- Docker Engine 容器化平台: Docker Engine 是一个开源的容器化平台，用于构建、发布和运行应用程序。它能够帮助开发人员将应用程序及其所有依赖项打包成一个标准化的单元。即容器，从而实现了应用程序的快速部署和可移植性。

- Portainer: Portainer是Kubemnetes、Docker、Docker Swam和Azure ACI的权威开源容器管理工具。它允许任何人部署和管理容器，而不需要编写代码。

- PostgreSQL 数据库: PostgreSQL 是一个功能强大的开源关系数据库管理系统，被广泛应用于企业级应用程序的开发和部署。它提供了高度的可扩展性、可靠性和性能，支持复杂的查询和事务处理。
- MySQL 数据库: 基于MariaDB Server 是最流行的开源关系数据库之一。 它由 MYSQL的原始开发人员制作，并保证保持开源。

- MyWebsQL 数据库管理工具: MyWebsQL是基于Web的MySQL数据库管理工具，它提供了一个简单而强大的界面，用于管理MySQL数据库。可以使用MyWebsQL来创建、删除、修改数据库和表，执行SQL查询，导入和导出数据等。

- TerraSync 文件同步工具: 通过 TeraSync(TS)应用程序，您可以将TNAS 打造成一台私有云盘服务器。TerraSync作为核心应用运行于 TNAS，配合 TeraSync Client(TSCient)、TerraSync Windows(内置于 TNAS PC for Windows)和 TerraSync mac(内置于 TNAS PC for mac)客户端，实现多设备间的文件同步。

- TerraSync Client 文件同步客户端: 通过 TeraSync(TS)应用程序，您可以将TNAS 打造成一台私有云盘服务器。TerraSync 作为核心应用运行于 TNAS，配合 TeraSync Client(TSClient)、TerraSync Windows(内置于 TNAS PC for Windows)和 TerraSync mac(内置于 TNAS PC for mac)客户端，实现多设备间的文件同步。

- Web Server: Web Server(Web 服务器)利用 TerraMaster Web Server，您可以将 TNAS 架设成一个网站服务器，通过 Web Server 的图形化界面设定多个虚拟主机,配置 PHP 、网页数据存储路径与网路协议，为网站维护管理提供便利。

- e107_cms 内容管理系统: e107是一个免费(开源)的内容管理系统，它允许您轻松地在线管理和发布您的内容。开发人员可以在构建网站和强大的在线应用程序时节省时间。用户可以完全避免编程!博客，网站，内部网一e107都能做到。如果在使用时界面显示不正常,可以在控制面板的安全模块中取消启用"使用HTTP内容安全策略(CSP)标头以提高安全性”

- elFinder 文件管理器: elFinder是一个用于 Web 的开源文件管理器，使用 jQuery Ul用javaScript 编写。Creation 的灵感来自于 Mac OS X操作系统中使用的 Finder 程序的简单性和便利性。

- TFM Backup: TFM(TerraMaster Folder Miror)是一个简单高效的备份工具。RAID1磁盘镜像能够提供数据冗余保护以避免磁盘故障导致的数据丢失，但也降低了磁盘空间的利用率。TFM 仅对存有重要数据的文件夹创建镜像备份而非整个磁盘空间，既能够提供数据保护，又能够增加磁盘空间的利用率。您甚至可以将一个远程文件夹挂载到本地，然后使用TFM 对远程文件夹的数据进行备份，或者将本地的数据备份到已挂载的远程文件夹。

- PHP 8.0 版本: PHP 8.0 是 PHP 语言的一个主要版本，于 2020 年 11 月 26 日发布。它引入了许多新功能和改进，包括性能提升、新的语法特性、更好的类型系统等。

- Deduplication Manager： 对于中等规模以上的企业来说，多个客户端存储的数据存在大量重复的内容，大大增加了存储空间的消耗，增加了企业的运营成本。TerraMasterDeduplication Manager (文件去重管理器)采用了非常直观人性化的用户界面，帮助企业 IT 管理人员轻松筛査存储设备中重复的文件，并对重复的文件采取自动或自定义的去重以达到节省服务器存储空间的目的。

- CloudSvnc： CloudSvnc 是一个云盘同步应用程序、用于在您的 TNAS 和云盘之间实现快速，安全的数据同步。CloudSyn 集合了多种云盘同步功能、包括 GoogleDrive，OneDrive,Amazon S3,Dropbox,Baidu,阿里云等，提供双向，单向多种云盘同步策略与同步日志监控等功能，是一个非常实用，高效的一站式云盘灾备解决方案。

- Loa Center(日志中心): Loa Center(日志中心)用于收集TOS 系统日志、文件服务日志、用户日志、部分网用日志等，有助于 TNAS 管理员皇握系统的运行状况。香系统异堂事件，以及时对系统异常进行处理。Log Center 也具备接受与发送日志功能，可将您的TNAS 作为日志服务器使用。

## YYC3 NAS（F4-423）的可安装应用列表，按功能分类呈现

### 一、备份类应用

- **TPC Backupper**：铁威马定制的Windows PC备份软件，随USB存储设备赠送。
- **Centralized Backup（中央备份）**：商业级专业备份工具，支持企业级数据集中备份。
- **CloudSync**：云盘同步工具，支持TNAS与主流云盘（百度网盘、阿里云等）双向同步。
- **Duple Backup Vault（双重备份服务端）**：实现TNAS间的跨设备备份。
- **Duple Backup（双重备份）**：TNAS本地数据灾备工具，支持备份与还原。
- **Snapshot（快照）**：基于BTRFS文件系统的灾备工具，支持共享文件夹/iSCSI快照。
- **TFM Backup（文件夹镜像）**：简单高效的备份工具，支持RAID 1磁盘镜像冗余。
- **TerraSync / TerraSync Client**：将TNAS打造为私有云盘服务器，支持数据同步。
- **USB Copy**：TNAS与USB存储设备间的数据备份工具（双向备份）。
