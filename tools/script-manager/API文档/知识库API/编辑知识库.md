# 编辑知识库

> 用于编辑已经创建好的个人知识库，仅传入要修改的字段。

## OpenAPI

````yaml openapi/openapi.json put /llm-application/open/knowledge/{id}
paths:
  path: /llm-application/open/knowledge/{id}
  method: put
  servers:
    - url: https://open.bigmodel.cn/api/
      description: 开放平台服务
  request:
    security:
      - title: bearerAuth
        parameters:
          query: {}
          header:
            Authorization:
              type: http
              scheme: bearer
              description: >-
                使用以下格式进行身份验证：Bearer [<your api
                key>](https://bigmodel.cn/usercenter/proj-mgmt/apikeys)
          cookie: {}
    parameters:
      path:
        id:
          schema:
            - type: string
              required: true
              description: 知识库id
      query: {}
      header: {}
      cookie: {}
    body:
      application/json:
        schemaArray:
          - type: object
            properties:
              embedding_id:
                allOf:
                  - type: integer
                    description: 知识库绑定的向量化模型ID
              name:
                allOf:
                  - type: string
                    description: 知识库名称
              description:
                allOf:
                  - type: string
                    description: 知识库描述
              background:
                allOf:
                  - type: string
                    description: 背景颜色，可选：blue, red, orange, purple, sky, green, yellow
              icon:
                allOf:
                  - type: string
                    description: 知识库图标，可选：question, book, seal, wrench, tag, horn, house
              callback_url:
                allOf:
                  - type: string
                    description: 回调地址（若修改向量模型，则需要重新构建知识，由客户决定是否单独配置回调）
              callback_header:
                allOf:
                  - type: object
                    description: 回调时header携带的k-v（若修改向量模型，则需要重新构建知识，由客户决定是否单独配置回调）
            required: true
            refIdentifier: '#/components/schemas/KnowledgeEditRequest'
        examples:
          example:
            value:
              embedding_id: 123
              name: <string>
              description: <string>
              background: <string>
              icon: <string>
              callback_url: <string>
              callback_header: {}
  response:
    '200':
      application/json:
        schemaArray:
          - type: object
            properties:
              code:
                allOf:
                  - type: integer
                    description: 响应码，200为成功
              message:
                allOf:
                  - type: string
                    description: 响应信息
              timestamp:
                allOf:
                  - type: integer
                    description: 响应时间戳
            refIdentifier: '#/components/schemas/KnowledgeEditResponse'
        examples:
          example:
            value:
              code: 123
              message: <string>
              timestamp: 123
        description: 业务处理成功
    default:
      application/json:
        schemaArray:
          - type: object
            properties:
              code:
                allOf:
                  - type: integer
              message:
                allOf:
                  - type: string
            refIdentifier: '#/components/schemas/LlmApplicationError'
        examples:
          example:
            value:
              code: 123
              message: <string>
        description: 请求失败。
  deprecated: false
  type: path
components:
  schemas: {}

````
