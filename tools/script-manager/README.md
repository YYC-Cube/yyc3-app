> 📋 **创建时间**: 2025-12-08 | **维护团队**: YYC3 AI Family

## 📖 系统概述

YYC3脚本综合管理系统是对所有分散在 `/Users/yanyu/www/` 目录下的脚本文件进行统一整理、分类、中文命名和管理的系统，旨在提供清晰的脚本组织结构和便捷的使用体验。

## 🎯 使用原则

### 命名规范
- 所有脚本使用中文命名，清晰易懂
- 文件名格式：`功能_描述.sh`
- 避免使用拼音和英文混合

### 功能保证
- 每个脚本都经过功能验证
- 保持原有脚本的依赖关系
- 确保脚本在移动后仍可正常使用

### 使用安全
- 重要操作提供确认提示
- 备份关键配置和数据
- 提供操作日志和错误处理

## 📖 使用指南

### 快速查找
1. 查看 `脚本索引.md` 了解所有脚本详情
2. 根据功能需求进入对应分类目录
3. 阅读脚本说明和依赖要求

### 执行脚本
```bash
# 赋予执行权限
chmod +x 脚本名称.sh

# 执行脚本
./脚本名称.sh

# 或者使用 bash 执行
bash 脚本名称.sh
```

### 自定义配置
- 修改脚本中的配置变量
- 根据环境调整参数设置
- 测试配置的准确性
