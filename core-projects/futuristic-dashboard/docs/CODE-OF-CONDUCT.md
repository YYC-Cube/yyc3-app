# YYC3 未来仪表盘 - 行为准则

## 我们的承诺

为了营造一个开放、友好的开发环境，YYC3未来仪表盘项目的参与者承诺，无论年龄、体型、残疾、种族、性别认同和表达、经验水平、国籍、个人外表、种族、宗教或性取向如何，都将不受骚扰地参与我们的项目和社区。

## 我们的标准

有助于创造积极环境的行为示例包括：

* 使用欢迎和包容性语言
* 尊重不同的观点和经验
* 优雅地接受建设性批评
* 专注于对社区最有利的事情
* 对其他社区成员表示同理心

参与者不可接受的行为包括：

* 使用性化语言或图像，以及不受欢迎的性关注或挑逗
* 恶作剧和人身攻击
* 公开或私下骚扰
* 未经明确许可发布他人的私人信息，如物理或电子邮件地址
* 其他在专业环境中被合理认为不适当的行为

## 我们的责任

项目维护者负责澄清可接受行为的标准，并应采取适当和公平的纠正措施来应对任何不可接受的行为。

项目维护者有权和责任删除、编辑或拒绝与本行为准则不符的评论、提交、代码、wiki编辑、问题和其他贡献，或者暂时或永久禁止任何行为不适当的贡献者。

## 范围

本行为准则适用于项目空间内的所有行为，包括在公共空间中代表项目或其社区时的行为。代表项目或社区的例子包括使用官方项目电子邮件地址、通过官方社交媒体账号发布或担任在线或离线活动的指定代表。

## 执法

如果您认为有人违反了行为准则，请通过指定的联系方式（[项目邮箱]）向项目团队报告。所有投诉将被及时和公平地审查和调查。

项目团队承诺对事件报告的调查保密。将采取适当的行动，这可能包括警告、临时禁令或永久禁令，取决于违规的性质和严重程度。
